      O aplikaci

      Movie Explorer vznikl jako závěrečná semestrální práce do předmětu 4IZ268 Webové technologie pro ZS24. Movie Explorer umožňuje vyhledávat filmy, zobrazovat o nich informace, přidávat je do seznamu oblíbených a disponuje historií vyhledávání.

      Responzivní design a zpracování chyb API je samozřejmostí.
