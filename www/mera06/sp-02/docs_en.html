OMDb Movie Finder Documentation

Author: Andreas Merhaut

Date: 9. 1. 2025, 12:00 PM

Introduction

The OMDb Movie Finder is a web application that allows users to search for movies and TV series, view details, and manage a list of favorites. It uses the OMDb API to fetch movie data dynamically via AJAX and offers a user-friendly interface.

Code Explanation

1. Search Section

The search section includes a form with inputs for the movie title and optional year. When the form is submitted, a query is sent to the OMDb API, and the results are dynamically displayed.

2. Results Section

The results section displays movies as cards with their title, year, and poster. Each card includes a "View Details" button, which fetches additional data for the movie.

3. Details Section

The details section shows detailed information about a selected movie or series, including the director, plot, genre, and poster. Buttons allow adding or removing the movie from favorites.

4. Favorites Section

The favorites section lists all saved movies with their title and year. Users can view details or remove movies from the list.

How to Use

  1. Enter a movie or series title in the search bar and optionally specify a year.
  2. Click "Search" to see results in the results section.
  3. Click "View Details" on any movie to view its detailed information.
  4. Add a movie to favorites using the "Add to Favorites" button or remove it if already added.
